# Encanto do Vale

Landing page em React + Vite + TypeScript para a doceria Encanto do Vale.

## Tecnologias

- React
- Vite
- TypeScript
- CSS customizado
- Tailwind CSS configurado

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Deploy

Projeto pronto para deploy estático na Vercel ou Render.
